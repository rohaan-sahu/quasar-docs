Quasar is a Solana program development framework for writing blazing fast SVM programs.

| Vault benchmark | **Quasar** | **Pinocchio** | **Anchor 0.32** |
| --- | --- | --- | --- |
| **Deposit CU** | **1,576** | - | 4,506 **(x2.86)** |
| **Withdraw CU** | **411** | - | 4,088 **(x9.95)** |
| **Binary size** | **5,848 bytes** | - | 185,384 bytes **(x31.7)** |
